This repository contains Python-based DevOps automation assignments.

## Assignment List

### Q1: Password Strength Checker
- Validates password strength using security best practices
- Checks length, uppercase, lowercase, digits, and special characters

### Q2: CPU Health Monitoring
- Continuously monitors CPU usage
- Alerts when CPU usage exceeds threshold
- Uses psutil library

### Q3: Configuration Management Automation
- Reads INI configuration file
- Extracts key-value pairs
- Stores configuration as JSON in SQLite database
- Provides GET API using Flask

### Q4: File Backup Automation
- Backs up files from source to destination
- Prevents overwriting using timestamped filenames
- Command-line based execution

## Technologies Used
- Python 3.x
- psutil
- Flask
- SQLite
- Git & GitHub
